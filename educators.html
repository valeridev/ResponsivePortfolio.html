‚òëÔ∏è Step-by-step Guide

1. Download Visual Studio Code (VS Code) and install it on your computer.

2. Open VS Code and create a new file named "index.html" (all lowercase, no spaces).

3. Copy and paste the code from Github into the new file in VS Code.

4. Customize the code to your preferences.

5. Save the file (Cmd + S on Mac, Ctrl + S on Windows).

6. Open Google Chrome and click "Open File" in the menu to select the index.html file you saved.

7. Create a new folder and drag & drop your index.html file into it.

8. Your personalized invitation webpage will be visible in your browser.


Make sure to save the code in VS Code and reload the webpage in Chrome after making any changes.
